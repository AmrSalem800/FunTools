# Project Summary
**Abqarino** is an AI-driven platform designed to enhance educational and professional experiences by generating historical events and conducting job interviews. It leverages advanced AI models to provide tools for historical data retrieval, interview simulations, and various creative applications, catering to individual users and educational institutions.

# Project Module Description
The project comprises several functional modules, including:

- **AI Service**: Interfaces with AI models to generate historical events, interview questions, and detailed model answers for user evaluation.
- **User Authentication**: Manages user login and session handling.
- **Data Management**: Handles database interactions for storing and retrieving user data and historical records.
- **Frontend Interface**: Provides a user-friendly interface for interaction with the application.
- **Creative Tools**: Features various AI-powered tools for creativity, including an excuse generator, personality analyzer, style time travel, emotion translator, what-if simulator, and fridge recipe generator.

# File Description Inventory
- **app/backend/README.md**: Documentation for backend setup and usage.

- **app/backend/main.py**: Main application entry point for backend services.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: TypeScript, React, Tailwind CSS
- **Database**: PostgreSQL
- **AI Services**: Atoms Cloud AI, Gemini API

# Usage
To set up the project:
1. Install backend dependencies:
   ```
   pip install -r app/backend/requirements.txt
   ```
2. Install frontend dependencies:
   ```
   pnpm install
   ```
3. Build the frontend:
   ```
   pnpm run build
   ```
4. Start the backend server:
   ```
   uvicorn app.backend.main:app --reload
   ```
